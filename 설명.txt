JIGAPServerStartLib -- Client와 Server에서 공통적으로 쓰는 함수와 객체들을 정의한 
라이브러리 입니다

--Server--
JIGAPServer - C++로 구현된 서버 입니다. 모든 서버 코드 여기에 구현되어있습니다

JIGAPServerCLR - C++로 구현된 서버를 WInform C# 에서 쓰기 위해 C++ 서버를 
랩핑한 프로젝트입니다.

JIGAPServerGUI - 서버의 GUI를 구현한 프로젝트입니다 프로그램의 시작점입니다.
모든코드가 이 프로젝트에서 돌아갑니다.

--client--
JIGAPClient - C++로 구현된 서버 입니다. 모든 클라이언트 코드 여기에 구현되어있습니다

JIGAPClientCLR - C++로 구현된 클라이언트를 WInform C# 에서 쓰기 위해 C++ 서버를 
랩핑한 프로젝트입니다.

JIGAPClientGUI - 클라이언트의 GUI를 구현한 프로젝트입니다 프로그램의 시작점입니다.
모든코드가 이 프로젝트에서 돌아갑니다.
